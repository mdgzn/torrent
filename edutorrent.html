Apa itu WebTorrent?

WebTorrent adalah implementasi protokol BitTorrent yang berjalan langsung di browser menggunakan WebRTC. Dengan teknologi ini, pengguna dapat berbagi file tanpa server penyimpanan.

  • Pengguna yang mengupload disebut Seeder
  • Pengguna yang mengunduh disebut Peer / Leecher
  • File diidentifikasi menggunakan Magnet URI
  • Transfer data terjadi langsung antar browser
