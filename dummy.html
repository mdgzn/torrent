Belajar Torrent P2P di Browser

Demo sederhana berbagi file menggunakan WebTorrent

📤 Seeder (Berbagi File)

Seeder adalah pengguna yang membagikan file ke jaringan torrent.

🔗 Magnet Link

Magnet link adalah identitas torrent yang digunakan untuk menemukan file di jaringan P2P.

📥 Downloader (Mengambil File)

Masukkan magnet link dari seeder lalu klik Join untuk mulai terhubung dengan peer lain.

📚 Panduan Penggunaan

  1. Klik Upload File untuk berbagi file asli dari komputer.
  2. Atau klik Dummy File untuk membuat file simulasi.
  3. Setelah file di-seed, sistem akan menghasilkan Magnet Link.
  4. Salin magnet tersebut ke pengguna lain.
  5. Pengguna lain dapat memasukkan magnet lalu klik Join Swarm.
  6. File akan diunduh langsung dari peer menggunakan teknologi P2P.
